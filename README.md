# Projet Admission Faculté de Médecine

## Description
Application web en PHP connectée à MySQL permettant de déterminer si un étudiant est admissible à la faculté de médecine.

## Fonctionnalités
- Formulaire de saisie
- Calcul de moyenne pondérée
- Enregistrement en base de données
- Affichage du résultat
- Liste des étudiants

## Technologies utilisées
- PHP
- MySQL
- HTML / CSS
- XAMPP

## Installation

1. Cloner le projet :
git clone URL_DU_PROJET

2. Mettre le dossier dans :
C:\xampp\htdocs

3. Créer la base de données MySQL

4. Lancer Apache et MySQL dans XAMPP

5. Ouvrir :
http://localhost/nom_du_projet
